# Strategic-Risk-Intelligence

A Streamlit-based financial analytics dashboard developed as part of my MBA Finance project.

## Overview

Strategic Risk Intelligence is a financial analytics platform designed to assess the credit risk and financial performance of Indian commercial banks using financial ratio analysis and data analytics techniques.

The dashboard enables users to evaluate bank performance, compare institutions, and generate data-driven insights through interactive visualizations and risk assessment models.

## Key Features

* Financial statement and ratio analysis
* Credit risk assessment and scoring
* Bank benchmarking and ranking
* Interactive dashboards and visualizations
* Comparative analysis of public and private sector banks
* Automated performance evaluation

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Excel

## Project Scope

* Analysis of 6 major Indian commercial banks
* Evaluation using key banking metrics including ROA, CAR, Net NPA, CASA, PCR, and Credit-Deposit Ratio
* Five-year financial performance assessment
* Data-driven risk classification and benchmarking
